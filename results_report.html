Q1: Hippocampal Ripple Density

Methodology & Validation

  • Filter Specs: 4th-order zero-phase bandpass Butterworth filter at 100–200 Hz.
  • Envelope: Hilbert transform magnitude, smoothed with a 10 ms moving window.
  • Normalization: Robust Z-scoring based on median and median absolute deviation (MAD).
  • Detection: Hysteresis thresholding starting at 4.0 Z-score, extending boundaries to 2.0 Z-score. Duration bounds restricted to 20–200 ms.
  • Artifact Rejection: Candidate events with raw signal amplitudes exceeding 20 robust SDs are discarded as high-frequency noise.

def detect_ripple_events(trace: np.ndarray, cfg: Config) -> tuple[int, float, np.ndarray]:
    # 1. 4th-order zero-phase bandpass Butterworth filter at 100-200 Hz
    sos = signal.butter(cfg.ripple_filter_order, [cfg.ripple_low_hz, cfg.ripple_high_hz], btype="bandpass", fs=cfg.fs_lfp, output="sos")
    filtered = signal.sosfiltfilt(sos, trace)
    
    # 2. Compute Hilbert envelope & smooth with 10 ms moving window
    envelope = np.abs(signal.hilbert(filtered))
    envelope = ndimage.uniform_filter1d(envelope, size=max(1, int(round(0.010 * cfg.fs_lfp))))
    
    # 3. Robust Median/MAD Z-score normalization
    envelope_z = robust_z(envelope)
    raw_z = np.abs(robust_z(trace))
    
    # 4. Hysteresis detection (Starts at 4.0 Z-score, extends to 2.0 Z-score)
    low = envelope_z >= cfg.ripple_end_z
    changes = np.diff(np.pad(low.astype(np.int8), (1, 1)))
    starts = np.flatnonzero(changes == 1)
    stops = np.flatnonzero(changes == -1)
    
    # 5. Restrict event duration bounds & reject high-amplitude noise (>20 robust SD)
    min_samples = int(math.ceil(cfg.ripple_min_ms * cfg.fs_lfp / 1000))
    max_samples = int(math.floor(cfg.ripple_max_ms * cfg.fs_lfp / 1000))
    
    centers = []
    for start, stop in zip(starts, stops):
        duration = stop - start
        if duration < min_samples or duration > max_samples: continue
        if np.max(envelope_z[start:stop]) < cfg.ripple_start_z: continue
        if np.max(raw_z[start:stop]) > cfg.ripple_artifact_z: continue
        centers.append((start + stop) / (2 * cfg.fs_lfp))
        
    return len(centers), len(centers) / usable_minutes, np.asarray(centers)
Q2: Within-Area Pairwise Spike-Train Interactions

Methodology & Validation

  • Spike counts: Computed in non-overlapping 100 ms bins.
  • Rate filter: Excluded all units firing below 0.05 Hz.
  • Correlation: Computed Pearson's $r$ between every within-area unit pair, taking the absolute value $|r|$ to measure total interaction strength.
  • Surrogates: Created 100 circular-shift surrogates per unit (shifted by 10-90% of duration) to isolate fast interactions from slow drifts or state shifts.
  • Aggregation: The interaction value per mouse is the mean observed $|r|$ minus the mean surrogate $|r|$.

Q3: Directed Functional Connectivity

Methodology & Validation

  • Spikes: Combined all within-area neuron spikes into area-level population trains.
  • Binning: Population spike trains binned at 2 ms resolution.
  • Detrending: High-pass filtered by subtracting a 100 ms Gaussian-smoothed trace.
  • Z-Score: Residual counts normalized to unit variance.
  • Asymmetry: CCG asymmetry ($\Delta r$) calculated as the mean difference $CCG(+lag) - CCG(-lag)$ for lags between 2 ms and 50 ms. Positive values denote that the source area leads the target.

def prepare_population_activity(counts: np.ndarray, cfg: Config) -> np.ndarray:
    # 1. 100 ms Gaussian smoothing filter for slow trends detrending
    activity = counts.astype(np.float32)
    sigma_bins = cfg.detrend_s / cfg.bin_s
    slow = ndimage.gaussian_filter1d(activity, sigma=sigma_bins, axis=1)
    
    # 2. Subtract slow trends & z-score residual high-frequency counts
    activity -= slow
    activity -= activity.mean(axis=1, keepdims=True)
    scale = activity.std(axis=1, keepdims=True)
    scale[scale == 0] = 1
    return activity / scale

def ccg_asymmetry(source: np.ndarray, target: np.ndarray, min_lag: int, max_lag: int) -> float:
    # 3. Sum lag correlation difference (positive - negative) over 2-50 ms
    differences = []
    for lag in range(min_lag, max_lag + 1):
        positive = np.mean(source[:-lag] * target[lag:], dtype=np.float64)
        negative = np.mean(source[lag:] * target[:-lag], dtype=np.float64)
        differences.append(positive - negative)
    return float(np.mean(differences))
